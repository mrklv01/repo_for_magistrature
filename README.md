# HR-Agent MVP

Инструмент предиктивного анализа рисков выгорания и перегрузки персонала на основе данных ITSM-системы.

Построен в рамках магистерской диссертации — Esil University, Астана, 2026.

## Быстрый старт (локально)

```bash
# 1. Установить зависимости
npm install

# 2. Добавить API-ключ Anthropic
cp .env.local.example .env.local
# Вставить ключ из console.anthropic.com

# 3. Запустить
npm run dev
```

Открыть [http://localhost:3000](http://localhost:3000).

## Использование

1. Нажмите **«Загрузить демо-датасет»** — или загрузите свою выгрузку CSV/XLSX из ITSM.
2. Проверьте сводку: количество тикетов, сотрудники, период.
3. Нажмите **«Запустить анализ рисков»** — ИИ проанализирует данные за 20–40 секунд.
4. Изучите дашборд: общий диагноз отдела, таблица сотрудников, клик по строке — детальная карточка.
5. Экспортируйте отчёт в PDF или HTML через кнопки в шапке.

## Требования к CSV/XLSX

Минимальный набор колонок (названия могут отличаться — система определит автоматически):

| Поле | Примеры названий |
|------|-----------------|
| Дата создания | `created_at`, `Дата создания`, `opened` |
| Принято в работу | `accepted_at`, `Принято`, `start_date` |
| Исполнитель | `assigned_to`, `Исполнитель`, `executor` |

Остальные поля (приоритет, категория, статус, описание) улучшают качество анализа, но не обязательны.

## Деплой на Vercel

```bash
npm i -g vercel
vercel --prod
```

В настройках проекта Vercel → Environment Variables добавить `ANTHROPIC_API_KEY`.
